新人友好度 — 哪类更容易突围

三因子模型: 签约新书密度×0.45 + 竞争分散度×0.30 + 排名可及性×0.25
签约新书密度（蓝）— 起点签约新书榜在该品类的占比，最直接的新书活跃度指标； 竞争分散度（绿）— 头部（Top 3）热度占比越低、市场越分散，新人越有机会； 排名可及性（紫）— 整体排名越靠前，榜单可及性越高，新人更容易被读者发现。 三项归一化后加权合成 0-100 的友好度评分。不依赖周间数据对比，仅用当前周截面数据。

分类动量 — 谁在风口之上

动量指数 = tanh(热度变化率/30)×0.6 + tanh(书目变化率/20)×0.4，归一化至 ±100
正值（绿）表示该分类本周总热度与书目数双增；负值（红）表示降温。使用 tanh 阻尼函数防止小品类因基数低而产生极端波动。

多维雷达 — 品类因子轮廓对比

雷达图对比各品类在 需求信号、竞争空间（1−头部集中度）、新人友好、增长动量 四个维度的表现。 覆盖面积越大、轮廓越均衡，说明品类综合机会越优。可点击图例筛选特定品类。
